# FUTURE_DS_03
# Marketing Funnel & Conversion Performance Analysis

## Project Overview

This project analyzes marketing funnel performance to identify conversion drop-offs, evaluate channel effectiveness, and uncover opportunities to improve lead-to-customer conversion rates. The dashboard provides actionable insights into customer acquisition and marketing efficiency.

## Dataset

* Marketing Funnel Dataset
* Marketing Channels: Google Ads, Facebook Ads, Instagram, Email, Organic Search
* Funnel Stages:

  * Website Visits
  * Product Views
  * Add to Cart
  * Checkout Started
  * Purchases

## Tools Used

* Power BI
* CSV Dataset
* Data Analytics

## Dashboard Features

* Total Website Visits KPI
* Total Purchases KPI
* Conversion Rate KPI
* Marketing Funnel Visualization
* Purchases by Channel
* Website Visits by Channel
* Conversion Rate by Channel
* Interactive Channel Filter (Slicer)

## Key Insights

* Significant drop-offs occur between funnel stages.
* Email marketing shows strong conversion performance.
* Google Ads generates the highest website traffic.
* Organic Search contributes consistent conversions with lower acquisition effort.

## Recommendations

* Optimize landing pages to increase visitor-to-lead conversion.
* Improve checkout experience to reduce abandonment.
* Invest more in high-converting channels.
* Retarget users who abandon the funnel before purchase.
